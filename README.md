# File Search Explorer

タブで探す、圧倒的な速さで見つかる。ローカル完結のファイル検索デスクトップアプリです。

Windows標準の検索よりも柔軟で、インデックス機能により2回目以降の検索が瞬時に終わります。

## ダウンロード

最新版は [Releases](../../releases) から入手できます。
ダウンロードするファイルは「FileSearchExplorer_v0.1beta.zip」です。
「Source code」と書かれた2つのリンクはGitHubが自動生成するもので、
アプリの実行には使用しません。

## 特徴

### タブで並行検索
複数のフォルダを同時に検索したいとき、タブを増やすだけ。Explorer のタブ感覚のまま、切り替えながら検索できます。

### インデックス機能
- 一度インデックスを作成しておけば、2回目以降の検索は瞬時
- 検索したフォルダは自動でバックグラウンドインデックス化
- 登録済みフォルダは起動時・30分ごとに自動で最新状態に更新
- 作業の邪魔をしない、静かなバックグラウンド処理

### 柔軟な検索オプション
- 部分一致・完全一致検索
- 拡張子フィルタ(画像・動画・文書・音楽・圧縮ファイルなどのカテゴリからワンクリックで追加可能)
- フォルダ名も検索対象にできる

### プライバシー第一設計
- ネットワーク通信は一切なし。オフライン環境でも実行可能
- インデックスデータベースはAES暗号化して保存。アプリを閉じると自動的に暗号化されます

### 言語
- インストーラーは日本語・英語対応
- アプリは日本語対応

## 動作環境

- Windows 11 (64bit) で動作確認済み
- Windows 10 (64bit) でも動作確認済み
- Pythonのインストールは不要です(単体で動作します)

## 注意事項

ダウンロード時に、ブラウザ(Chrome・Edgeなど)が「このファイルは危険な可能性があります」といった警告を表示することがあります。これは新しく・利用者の少ないファイルに対してブラウザが表示する一般的な確認であり、ウイルスが含まれているという意味ではありません。「詳細」→「保持する」(または「このまま残す」)を選択してダウンロードを続行してください。

初回起動時に「WindowsによってPCが保護されました」といった警告が表示されることがあります。これは未署名の個人配布アプリで一般的に見られる確認画面です。「詳細情報」→「実行」で起動できます。

ウイルス対策ソフトによる誤検知が発生する場合があります。使用しているセキュリティソフトごとに回避方法が異なりますので、お使いのソフト名と合わせて検索し、回避方法をご確認ください。
